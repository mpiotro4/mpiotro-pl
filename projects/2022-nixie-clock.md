## PL

Projekt zegara wykorzystującego klasyczne lampy nixie do wyświetlania czasu.

### Opis projektu

Zegar Nixie to połączenie retro estetyki z nowoczesną elektroniką. Wykorzystuje oryginalne lampy nixie z lat 60-tych do wyświetlania godziny w unikalny, ciepły sposób.

### Funkcjonalności

- Wyświetlanie godziny na lampach nixie
- Synchronizacja czasu przez Internet (NTP)
- Automatyczna regulacja jasności
- Zabezpieczenia wysokiego napięcia

### Technologie

- Arduino/ESP8266
- Projektowanie układów wysokiego napięcia
- Projektowanie PCB
- Programowanie mikrokontrolerów

### Wyzwania techniczne

Największym wyzwaniem było zaprojektowanie bezpiecznego układu zasilania wysokiego napięcia (170V) wymaganego przez lampy nixie oraz integracja z niskonapięciową logiką mikrokontrolera.

## EN

A clock project using classic nixie tubes to display time.

### Project Description

The Nixie Clock combines retro aesthetics with modern electronics. It uses original nixie tubes from the 1960s to display time in a unique, warm way.

### Features

- Time display on nixie tubes
- Internet time synchronization (NTP)
- Automatic brightness adjustment
- High voltage safety features

### Technologies

- Arduino/ESP8266
- High voltage circuit design
- PCB design
- Microcontroller programming

### Technical Challenges

The biggest challenge was designing a safe high voltage power supply (170V) required by nixie tubes and integrating it with the low-voltage microcontroller logic.
