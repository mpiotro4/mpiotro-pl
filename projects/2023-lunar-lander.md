## PL

Implementacja klasycznej gry Lunar Lander z wykorzystaniem Python i Pygame.

### Opis projektu

Lunar Lander to gra zręcznościowa, w której gracz musi bezpiecznie wylądować statkiem kosmicznym na powierzchni księżyca, zarządzając paliwem i siłą ciągu.

### Funkcjonalności

- Realistyczna fizyka lotu
- System zarządzania paliwem
- Różne poziomy trudności
- System punktacji

### Technologie

- Python 3
- Pygame
- Fizyka 2D

## EN

Implementation of the classic Lunar Lander game using Python and Pygame.

### Project Description

Lunar Lander is an arcade game where the player must safely land a spacecraft on the moon's surface while managing fuel and thrust.

### Features

- Realistic flight physics
- Fuel management system
- Different difficulty levels
- Scoring system

### Technologies

- Python 3
- Pygame
- 2D Physics
