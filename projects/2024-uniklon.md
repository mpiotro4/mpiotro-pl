## PL

Projekt klonu legendarnego syntezatora Unison Research.

### Opis projektu

Uniklon to mój projekt odtworzenia klasycznego syntezatora analogowego. Projekt obejmuje:

- Projektowanie i budowę analogowych obwodów
- Kalibrację i testowanie dźwięku
- Integrację wszystkich modułów

### Technologie

- Elektronika analogowa
- Obwody audio
- Projektowanie PCB

### Cele projektu

1. Odtworzenie oryginalnego brzmienia
2. Modernizacja niektórych komponentów
3. Stworzenie w pełni funkcjonalnego instrumentu

## EN

A clone project of the legendary Unison Research synthesizer.

### Project Description

Uniklon is my project to recreate a classic analog synthesizer. The project includes:

- Designing and building analog circuits
- Sound calibration and testing
- Integration of all modules

### Technologies

- Analog electronics
- Audio circuits
- PCB design

### Project Goals

1. Recreate the original sound
2. Modernize some components
3. Create a fully functional instrument
